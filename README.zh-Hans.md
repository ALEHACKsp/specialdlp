### Summary
所有 Strict DLP Chinese(SDC) 基于 eMule Xtreme 官方 DLP 的严格版本。在 eMule Xtreme 的官方 DLP 的基础上，SDC 将 VeryCD-Mod 和 VeryCD-EasyMule-Mod (第1版) 中的部分或全部加入了 SoftBan 列表，原因是 GPL-breaker、私有网络和社区客户端等不良行为。
