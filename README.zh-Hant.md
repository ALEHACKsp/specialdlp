### Usage

### Summary
所有 Strict DLP Chinese(SDC) 基於 eMule Xtreme 官方 DLP 之嚴格版本。於 eMule Xtreme 的官方 DLP 的基礎上，SDC 將 VeryCD-Mod 和 VeryCD-EasyMule-Mod (第1版) 中的部分或全部加入了 SoftBan 列表，原因是 GPL-breaker、私有網路和社群客戶端等不良行為。
